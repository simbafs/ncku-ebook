# 怎麼找書？
通常書名丟到 Google 都會找的到，但是可能要錢，但都會有 ISBN，記下來。丟去 [https://1lib.tw](https://1lib.tw) 搜尋，這樣可以最快速度找到你要的書。用 ISBN 找事最快的
